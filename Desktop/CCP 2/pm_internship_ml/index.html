Semantic Search

Unlike traditional keyword matching, our AI understands the meaning behind your query. Search for "web development" and discover "full-stack", "frontend engineer", or "React developer" roles automatically.

Location Intelligence

When no internships are found in your preferred city, the system automatically discovers relevant opportunities in nearby locations, intelligently sorted by distance.

Multi-Field Analysis

Our AI analyzes multiple dimensions of each internship:

  • Job title and role requirements
  • Required skills and technologies
  • Job description and responsibilities
  • Company profile and category

Relevance Scoring

Every result includes a match percentage (0-100%) showing how well it aligns with your search criteria, helping you prioritize your applications effectively.

The Problem

Traditional internship search relies on exact keyword matching, often missing highly relevant opportunities. Location-based searches frequently return empty results when a specific city has no listings.

Our Solution

This project enhances the PM Internship Scheme search with cutting-edge machine learning. Using sentence transformers and cosine similarity, we deliver smarter, more contextual, and highly relevant results.

Key Features

  • Semantic understanding of natural language queries
  • Automatic nearby location suggestions
  • Relevance-ranked search results
  • Multi-field weighted analysis

Technology Stack

Built with Python, Flask, and Sentence Transformers (all-MiniLM-L6-v2). The modern frontend connects to a local API that processes searches in real-time with lightning-fast response times.
